Crear repo en GIT.

1. Abrir la carpeta donde vayan a trabajar . Ya sea con 
bash o con VSCode.

2. Agregar primer comando: git initial

3. Agregar todos los archivos: git add.

4. Verificacion de status, siempre que quieran ya que solo
es informativo: git status 

5. Confirmar los cambios con: git commit -m "Algun 
mensaje descriptivo"

Crear ramas: 
1. Saber en cual rama estoy: git branch

2. Crear nueva rama: git branch nombreDeRama.

3. Trabajar lo que necesitemos en la rama, hacer add y
commit
